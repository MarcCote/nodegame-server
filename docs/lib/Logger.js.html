¶ Logger.logText

Default log function for text

Logs to stdout and / or to file depending on configuration.

Params
text string mixed The string to log. If not a string, it is stringified
level string number The log level for this log
meta object Additional meta-data to be logged. meta.name is automatically added, if the logger has a name.
See
Winston 
Logger.prototype.logText = function(text, level, meta) {
    if ('string' !== typeof text) text = JSON.stringify(text);
    meta = meta || {};
    if (this.name) meta.name = this.name;
    this.logger.log(level || 'info', text, meta);
};
